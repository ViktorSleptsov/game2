# Как скачать калькулятор

Нажать либо code на странице калькулятора и выбрать download zip, либо скопировать ссылку на страницу и через командную строку ввести git clone <ссылка на страницу>

# Инструкция по использованию калькулятора.

Версия Python - 3.10 и выше.

Запускать через командную строку или похожие приложения.

Запускать через команду py и затем уже полное название программы.

# Функционал

Сложение, вычитание, умножение, деление, квадрат и корень квадрата присутствуют в данном приложении
